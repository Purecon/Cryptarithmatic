Program Cryptarithmatic
Tugas Kecil 1 Strategi Algoritma

File source code di \src, Tucil1.py
File executeable di \bin, Tucil1.exe
Program menggunakan brute force dengan bahasa python 3.7.3. 
Program dapat memecahkan masalah cryptarithmatic dengan algoritma brute force.
Tidak menggunakan library kecuali time dan os sehingga tidak ada dependency atau packet yang perlu diinstall.

Cara Menggunakan:
1.Run Tucil1.exe di dalam folder bin untuk menjalankan executeable atau dengan interpreter python jalankan Tucil1.py di src
2.Masukkan nama file sesuai dengan tes yang akan dijalankan (pastikan terdapat folder \test dengan file yang akan dibuka)
3.Tunggu program sampai keluar output setelah program berhasil berjalan.

Disclaimer: Waktu run dengan executeable lebih lambat dari hasil test dengan interpreter
